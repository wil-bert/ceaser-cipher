# Ceasar Cipher
This application is used to encrypt and decrypt letters. It uses a technique called ceasar cipher that was used back in the day by Ceasar.

## Author
Ian Wilbert

## Prerequisites
1. Java
2. Gradle

## Setup
* Clone the repo
* Compile gradle
* Run directories then run "java App"

## Testing
